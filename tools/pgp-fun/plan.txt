.plan

Man wähle mehrere m_i, so dass m_i = M * ~m_i.

(~m_i viel kleiner als n)

Dann gilt fuer sig(m_i)

    sig(m_i) = sig(M) * sig(~m_i)

Hoffnung ist, dass man so eine Menge 

    ( ~m_i , sig(~m_i) )-Pärchen

hat, so dass man eine sinnvolle Wahrscheinlichkeit hat fuer eine zu signierende Nachricht N 
eine Zerlegung

  N = ~m_i * ~m_j * ~m_k .... 

zu finden. Dann:

  sig(N) = sig(~m_i) * sig(~m_j) * sig(~m_k) ...
