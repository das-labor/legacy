Das Layout für den Laserbelichter ist nicht spiegelverkehrt, das für die Folien schon.

Im Endeffekt muss das Layout so herum auf die Platine, das man die Schrift lesen kann, wenn man auf die Kupferseite der Platine schaut.

Erstellung der Aetzvorlage:

In eagle nur die Ebenen bottom, pads, vias, dimension und center-drill anschalten. Dann das Ganze in ein pdf drucken. Gespiegelt für den Laserbelichter, sonst ungespiegelt.
Um mehrfach-layouts zu erstellen kann man in eagle den Schaltplan schließen, und das Board mehrmals nebeneinander kopieren. Das Board darf man dann aber nicht ausversehen speichern.
